# Sugestão de postagem para LinkedIn

Desenvolvi uma plataforma DevOps de monitoramento ambiental em tempo real, integrando dados públicos de clima e qualidade do ar em uma arquitetura completa com Docker, FastAPI, PostgreSQL, Streamlit, Prometheus, Grafana e GitHub Actions.

O projeto coleta dados reais da web, processa indicadores ambientais, classifica níveis de risco e exibe tudo em um dashboard de fácil visualização.

Principais pontos técnicos:

- Coleta automática de dados em tempo real
- API REST com FastAPI
- Banco PostgreSQL para histórico
- Dashboard interativo com Streamlit
- Observabilidade com Prometheus e Grafana
- Pipeline CI/CD com GitHub Actions
- Execução completa com Docker Compose

Este projeto une DevOps, Engenharia de Dados, Observabilidade e ESG em uma aplicação prática, visual e preparada para apresentação a recrutadores.
