# Guia para Recrutadores

Este projeto demonstra uma plataforma DevOps com ingestão de dados reais em tempo real, persistência, API, dashboard, observabilidade e pipeline CI/CD.

## Competências demonstradas

- Linux e automação de ambiente.
- Docker e Docker Compose.
- Backend com FastAPI.
- Data Engineering com coleta externa em tempo real.
- Banco PostgreSQL com modelagem e índices.
- Observabilidade com Prometheus e Grafana.
- CI/CD com GitHub Actions.
- Boas práticas: healthcheck, retry, logs, testes e lint.

## História do projeto

A plataforma monitora cidades brasileiras usando dados públicos de clima e qualidade do ar. O objetivo é demonstrar como dados externos podem alimentar uma arquitetura confiável, visual e pronta para ambientes reais.
